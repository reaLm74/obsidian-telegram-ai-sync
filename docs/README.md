## üÜò Getting Help

### Before Asking for Help
1. **Check Documentation**: Search these guides for your specific issue
2. **Review Settings**: Verify your configuration matches the guides
3. **Test Basic Setup**: Ensure basic functionality works before advanced features
4. **Check Logs**: Look for error messages in Obsidian's developer console

### How to Get Support
1. **GitHub Issues**: For bugs, feature requests, and technical problems
2. **Telegram Contact**: For quick questions and general support
3. **Community**: Share experiences and get help from other users

### Providing Information
When asking for help, please include:
- Plugin version and Obsidian version
- Relevant error messages or logs
- Steps to reproduce the issue
- Your configuration (without sensitive tokens)
- Expected vs actual behavior
